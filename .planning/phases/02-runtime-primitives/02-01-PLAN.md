**Part B: Add FieldAccess arm to infer_expression (per D-03)**

In `src/parser/infer.rs`, replace the `_ => None` catch-all (line 136) with an explicit `HExpression::FieldAccess { expr, field }` arm before the catch-all:

```rust
HExpression::FieldAccess { expr, field } => {
    if let HExpression::FunctionCall { fn_name, .. } = field.as_ref() {
        // Method call: obj.method() — resolve receiver type, look up mangled name in symbol table
        if let Some(Type::Struct(struct_name)) = infer_expression(expr, table, var_map) {
            let mangled = format!("{}_{}", struct_name, fn_name);
            return infer_function_call(&mangled, table);
        }
    }
    None
}
```

Keep `_ => None` after this new arm.
